Be in touch with users opinion

View Review provides live reviews about apps, movies, albums and books in Google Play.

Each item of RSS or JSON response contains Rating, Author, Title, Comment and Date of new review.

API Documentation

It is so simple!
Just copy "id" parameter from the URL of item on Google Play page, and paste it in the URL of the feed.

For example, if you would like to track reviews of Clash of Clans, your RSS URL should be
http://playparser-inmycloud.rhcloud.com/rss/app?id=com.supercell.clashofclans

and JSON
http://playparser-inmycloud.rhcloud.com/json/app?id=com.supercell.clashofclans



The general URL pattern is:

http://playparser-inmycloud.rhcloud.com/{format}/{type}?id={item id}

Where {format} is "rss" or "json", and {type} is "app", "album", "movie" or "book"

  • You also can provide "hl" and "limit" parameters to set language and max number of reviews.
  • Filter reviews by rating with "from" and "to" params (from 1 to 5 inclusive).
  • For JSON format the "page" parameter can control the page number you would like to see in results.

Integrate it with IFTTT

Oh my! It is so simple to be in touch with IFTTT!

IFTTT provides a really neat way to integrate everything with everything! You can use this awesome tool to get new reviews and ratings as fast as possible.

Just create a new recipe with RSS trigger and point it to RSS URL of your feed. It will react on new items in feed, notifying your Pushbullet, Smart watches, creating todos in Todoist and much more.
